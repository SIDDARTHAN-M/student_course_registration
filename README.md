# Course Registration System

A simple course registration system built using Python (console-based).

# Overview

A console-based Course Registration System developed using core Python concepts. This project simulates a simple student registration process where users can view available courses, register, modify their selection, and confirm their registration.

# Key Features

- Display available courses with numbered options
- Register using Registration Number and Name
- Select course using number (no typing errors)
- Modify selected course before final submission
- Confirm and complete registration
- Clean and formatted output display

# How to Run

1. Install Python (if not already installed)
2. Save the program as `student_course_registration.py`
3. Open terminal or command prompt
4. Run the program:
    ```bash
    python student_course_registration.py
    ```
5. Follow on-screen instructions

# Note

This program is a basic Python console application. It does not use databases or file storage, so data is not saved permanently.
